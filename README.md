# 🌍 Countries App

A responsive, interactive web app that displays information about countries using the [REST Countries API](https://restcountries.com/). Users can:

- Browse all countries
- Filter by region
- Search by name
- View detailed information on each country
- Explore border countries
- Enjoy a seamless experience with shimmer (skeleton) loading effects

---

## 🚀 Features

- 🧭 Search by country name
- 🌐 Filter by region (Africa, Americas, Asia, Europe, Oceania)
- 🗺 View detailed country info including:
  - Native Name
  - Population
  - Region / Subregion
  - Capital
  - Top-level domain
  - Currencies
  - Languages
  - Border countries (linked)
- ✨ Shimmer loader on page load for better UX
- 🌓 Responsive design (mobile-first)

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- [REST Countries API](https://restcountries.com/)
